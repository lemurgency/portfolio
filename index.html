Skeeball

My brother and I joined a skeeball league when I first arrived in Austin. The league currently uses pencil and paper to keep scores and someone has to manually enter all the data into the database. My goal was to build an app that players could use on their phone to track scores. There is much work to be done, but this is the base app.

The app uses Angular for the front end and a Rails API for the backend. Here is the Frontend and Backend.

RSVPeanut

RSVPeanut was a mapping application that queries 3 APIs including Eventbrite, SongKick, and Meetup to gather local events. I worked on the backend and helped construct a RESTful Rails API that our Ember frontend would query. I also helped construct API calls to the 3 event services and created optimally designed return values. The code can be viewed here.

Music Snob

Inspired by hours of mismatched music being played during our MakerSquare work sessions, we decided to build an app that would allow you to build collaborative playlists on Spotify for our 3 day Hackathon project. We used Meteor with MongoDB to build our playlists that were made up of user suggested songs. Music Snob can be viewed Here. I personally built most of the controller and controller helper methods as well as much of the model manipulation. These methods were used to route, filter, and load songs.
